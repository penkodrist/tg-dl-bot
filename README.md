# Changelog
## 0.1.0
- добавлена логика для ДБ
  - дб способно запоминать настройки для отдельных каналов и чатов
  - есть функционал для перезаписи данных
  - дб возвращает акутальные данные для каналов в некоторых командах
- изменена логика отправки сообщений. Теперь все действия бота выведены в отдельную фукнцию. Появлиась возможность для удобного расширения функицонала
- теперь пользователь не может пользоваться ботом до прописывания команды /start.
  - бот не будет выдавать одно и то же сообщение на /start, если он был синициализирован для канала.
- иные мелкие изменения в коде, о которых я не помню
